Cara Daftar Akun Midtrans:

Buka dashboard.midtrans.com → klik Register
Isi data: nama, email, nomor HP, nama bisnis
Verifikasi email yang dikirim Midtrans
Login → akan masuk ke mode Sandbox (uji coba) secara default
Ambil API Key (Sandbox dulu):

Di dashboard, klik Settings → Access Keys
Salin:
Server Key → SB-Mid-server-xxxx
Client Key → SB-Mid-client-xxxx
Masukkan ke sistem Pabalu melalui admin → Akun Owner → konfigurasi Midtrans owner yang bersangkutan
Upgrade ke Production (agar bisa terima uang sungguhan):

Di dashboard Midtrans → Activation → lengkapi dokumen bisnis:
KTP pemilik
NPWP (jika ada)
Rekening bank
Dokumen legalitas usaha (SIUP/NIB jika ada)
Tunggu verifikasi Midtrans (1–3 hari kerja)
Setelah aktif, ambil Production Key (tanpa prefix SB-) dari menu Settings → Access Keys dengan toggle Production
Update key di sistem dan aktifkan Mode Produksi di konfigurasi owner
